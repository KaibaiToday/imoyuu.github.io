认识结构

【第1题】
2008年北京奥运会的主会场--"鸟巢”，其主体结构类型属于（)

A　框架结构
B　实体结构
C　以上都不是
D　壳体结构
A B C D
答案：A
实体结构可看作实心结构，空心的鸟巢不算; 壳体结构比如头盔，各种设备的外壳。通常外壳是"层"状的，就是一个面（想一想头盔光滑的面）。鸟巢属于框架结构，钢筋支架构成了框架（一条条纵横交错的线组成)。


【第2题】
我们生活中常见的水库的大坝、长城的城墙等结构，主要依靠自身来承受各种力的作用。这种结构类型属于()

A　壳体结构
B　实体结构
C　框架结构
D　组合结构
A B C D
答案：B
本题的考点为结构的类型。根据物体的结构形态在受力时承受和传递力的 方式的差别，将结构分为实体结构、框架结构和壳体结构三种基本类型。实体结构通常是指结构体本身是实心的结构，它的受力特点是，外力分布在整个体积中，如实心墙、大坝等;框架结构通常是指结构体由细长的构件组成的结构，其受力特点是，支撑空间而不充满空间，如铁架塔、建筑用脚手架、厂房的框架等;壳体结构通常是指层状的结构，它的受力特点是，外力作用在结构体的表面上，如摩托车手的头盔、飞机的外壳、贝壳等。 根据题干，“水库的大坝、长城的城墙等结构，主要依靠自身来承受各种力的 作用”，符合实体结构的定义，故选择“实体结构”。


【第3题】
常用的工具框锯是由工字形木框架、绞绳、绞片、锯条等组成。锯条两端用旋钮固定在框架上，绞绳绞紧后，锯条被绷紧即可使用,此时锯条的主要受力形式是()

A　剪切力
B　压力
C　拉力
D　扭转力
A B C D
答案：C
本题的考点为构件的受力形式，基本受力形式有拉力、压力、剪切力、扭转力和弯曲力。拉力，指物体所承受的拉拽力;压力，指挤压物体的力;弯曲力，指作用于物体，使它产生弯曲的力;剪切力，指两个距离很近，大小相等，方向相反，且作用于同一物体.上的平行力;扭转力，指反方向向物体两端均匀施力，使物体发生扭转形变的力。本题中，框锯框架的左端在绞绳的作用力下向里收缩，导致连接锯条的框架右端向外拉，所以锯条受到的外力是向外的拉拽力，故选择“拉力”


【第4题】
北京奥运会的主会场--"鸟巢"主体结构类型属于（）

A　框架结构
B　以上都不是
C　实体结构
D　壳体结构
A B C D
答案：A
本题考查常见结构的类型，根据物体的结构形态，结构分为三种基本类型∶实体结构-;通常是指结构体本身是实心的结构;框架结构∶通常是指结构体本身由细长的构件组成，支撑空间却不充满空间;壳体结构∶通常是指层状的结构。"鸟巢"的结构是由细长的构件组成支撑起空间的结构，所以是框架结构。

A　扭转力
B　弯曲力
C　压力
D　剪切力

答案：A

【第6题】
摩托车事故中驾乘人员死亡的70%以上，都是头盔问题造成头部受伤所致。质量合格的安全头盔能有效保护人的头部，是因为（）

A　合格头盔的密封性好，里面的空气能起到缓冲撞击的作用
B　合格头盔的强度大，可以避免交通事故的发生
C　当合格头盔受到外力冲击时，外力将由弧面整体分担承受，抵抗变形
D　合格头盔戴在头部空隙小，头盔是一个坚固的实体结构
A B C D
答案：C
壳体结构通常是指层状的结构。它的受力特点是，外力作用在结构体的表面上，如摩托车手的头盔、飞机的外壳、贝壳等。实体结构通常是指结构体本身是实心的结构。它的受力特点是，外力分布在整个体积中，如实心墙、大坝等。合格头盔的强度虽然大，但不能避免交通事故的发生，它只是能避免外力对头骨的撞击。当出现交通意外时，合格头盔的某一部位受到一个大的撞击力，由于头盔的壳体结构，受到外力作用时会有整个弧面承担，使得所受力迅速分布到其表面，形成整个头盔表面均匀受力，这是对外力的第一次减弱。合格头盔内部还有一柔性减震结构，它可以减弱头盔壳体上的力向人脑壳的传递，这样就起到了保护摩托车驾驶者安全的作用，并不是因为它的密封性好，里面的空气能起到缓冲撞击的作用。

【第10题】
图示为一款机械手。在力F作用下，拉杆通过连杆带动夹持手绕着销轴转动，将工件夹紧。连杆和夹持手的主要受力形式是（）



A　连杆受拉、夹持手受弯曲
B　连杆受压、夹持手受压
C　连杆受压、夹持手受弯曲
D　连杆受弯曲、夹持手受弯曲
A B C D
答案：C
技术素养(物化能力),学科内容(认识结构的受力)。 一个设计合理的结构，应该能承受正常范围内，外界的各种作用力，能抵抗各种变形。要清楚在这些力的作用下，构件能否安全、可靠地工作，就需要分析构件的承载能力。根据构件的受力和变形形式，可以将构件的受力分为受拉、受压、受剪切、受弯曲、受扭转五种基本形式。在分析构件受力时，要明确结构的状态，每个构件的受力效果及形变情况，并且要关注力与力之间的相互关系，表现在结构上，其形态变化也应该是相互的。当拉杆受力F作用后，连杆会随力的方向运动，连杆外的作用支点受力向内，导致其构件变短而受压，夹持手受三个点的力，受弯曲。


【第11题】
