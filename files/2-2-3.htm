【第1题】
知识无界限，通用技术学科和信息技术学科也有相结合的地方。下图所示为某程序流程图片断，若i和s的初值均为0，则该程序段结束后，s的值为( )



A　1到101之间的整数和
B　2到101之间的整数和
C　2到100之间的整数和炉温
D　1到100之间的整数和
A B C D
答案：A
解析： 此题难点在于确定i的最大值是100还是101。因为i=i+1，在s=s+i的前面，所以当ⅰ=100时，下行，执行ⅰ=i+1后，i=101，s=s+101，所以101这个数是可以计算在内的。所以s的值是1到101之间的整数和。
