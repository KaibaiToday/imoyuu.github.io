第三章 系统及其设计

系统的优化

【第1题】
高速公路时在经过居民区的路段,要建造隔音墙,以减少噪音污染,优化居民区的生活环境。以下不属于影响系统优化因素的是（ ）

A　隔音墙的高度
B　音墙的形状
C　隔音墙的材料
D　噪音的传播速度
A B C D
答案：D
本题考査影响系统优化的因素。系统优化的方法要求比较容易实施,隔音墙的髙度、材料、形状容易控制,而噪音的传播速度不可控制,所以不属于影响系统优化的因素


【第2题】
苗圃智能灌溉系统功能很强大。系统根据监测到的气象、土壤、植物等信息，控制灌溉设备工作，使土壤保持合适的湿度，从而达到节水节肥的目的。为了提高该系统的性能，需要进一步优化。下列不属于影响该系统优化的因素是（）

A　对气象、土壤、植物的监测精度
B　灌溉设备的产品质量
C　天气和植物的生长特性
D　无线网络的传输质量
A B C D
答案：C
本题考察的是影响系统优化的因素，能人为调节的叫"约束条件"，能人为调节并对系统目标有直接影响，叫"影响因素，天气和植物的生长特性，是不可改变因素，是不能人为调节的。


【第3题】
系统中影响系统优化的因素，是指对系统的结果产生显著影响并且可以人为调节的因素。山东某地区因地制宜采取了“麦、棉、瓜、玉米”的套种模式。在正常天气条件下，按照套种的技术要求操作，平均每亩收成比原先提高了百分之五十。在此农作物种植系统的优化中，影响因素是（）

A　套种技术
B　农作物的生长特性
C　气候
D　天气
A B C D
答案：A
由题可知，影响系统优化的因素是指对系统的结果产生显著影响并且可以人为调节的因素。由这点我们可以排除气候、农作物生长特性，因为它们不可以人为调节;题目中说在正常天气下，按照套种的技术要求操作提高了百分之五。而天气无法判断为是否人为控制，而套种技术的成败影响收成。


【第4题】
列车提速是对铁路运输系统的优化。以下说法正确的是( ）。

A　只要有先进的机车，就能提速
B　只有重新铺设铁路，才能提速
C　只有使用"动车组"才能提速
D　不仅需要有先进的机车，还需要道路、桥梁、隧道等基础设施的配套改进
A B C D
答案：D
系统优化需要条件，包括内部条件和外部条件
