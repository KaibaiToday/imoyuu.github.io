探究结构

【第1题】
地震发生时，第-时间老师会让学生迅速猫腰钻到课桌下面，防止被落物砸伤。如果需要进一步增加课桌的强度，下列措施合理的是()

A　减少桌面厚度
B　提高桌腿高度
C　改变桌腿倾斜度,扩大支撑面
D　采用高强度材料
A B C D
答案：D
本题的考点为结构的强度。结构的强度是指结构具有的抵抗被外力破坏的能力，它与结构的形状、使用的材料、构件之间的连接方式等因素有密切的关系。结构的稳定性是指结构在负载的作用下维持其原有平衡状态的能力，影响因素主要有重心位置的高低、结构与地面接触所形成的支撑面的大小和结构的形状等。“采用高强度材料”，是结构的强度有关的因素，故该选项是正确的。“减少桌面厚度”减小了构件的横截面积，属降低结构强度。其余两个选项属结构稳定性相关因素。

【第6题】
小明设计了一款造型美观的照片树，“树枝上”布满了小夹子，可以用来夹照片、纸张等。为了提高此产品的稳定性，下列措施中最可行的是（）

B　增加“树枝”和夹子的数量

答案：A

【第12题】
图示为一款木制衣帽架。为了提高其稳定性下列措施不合理的是（）



A　缩短衣帽钩伸出的长度
B　增大底座的支撑面积
C　缩小立柱的截面积
D　降低衣帽架的高度
A B C D
答案：C
技术素养(创新设计、物化能力),学科内容(结构的稳定性)。 结构的稳定性是指结构在荷载作用下维持其原有平衡状态的能力。如果一个物体的结构不能有效地抵御荷载的作用，那么该物体就很难保持原有的平衡状态。降低重心位置、增大支撑面、设计三角形，以及保持重心位置在支撑面内都是有效提高结构稳定性的办法。缩小立柱的截面积并没有提高结构的稳定性，所以不合理。同时，在考虑结构稳定性时还要同步考虑结构的功能。


【第13题】
如图所示为一款教学用摄像头展台，需要使用时打开拍摄横杆，以下关于摄像头展台说法不正确的是：（）





A　增大支撑杆的横截面积，可以加强结构的强度
B　工作时拍摄横杆主要受弯曲
C　增大底座面积，可以加强结构的稳定性
D　为保证结构强度，拍摄横杆和支撑杆要采用刚连接
A B C D
答案：D
影响结构强度的因素有材料、结构的形状和结构的连接方式。在具体的技术领域中，连接方式存在多种形式，不同的连接方式、结构形式适用于不同强度要求的场合.刚连接是指被连接的构件在连接处既不能相对移动，也不能相对转动。题目中的摄像头展台的拍摄横杆是需要可以折叠的，所以不能用刚连接，用铰连接也完全可以满足强度的需要。铰连接是指被连接的构件在连接处不能相对移动，但可相对转动

B　增加支撑柱的高度

本题答案为：C。影响结构稳定性的因素主要有:重心高低、支撑面的大小、结构的形状等。一般是通过降低重心，增大支撑面，改变结构的形状来增加结构的稳定性。增加灯的数量灯更亮，增大太阳能板的面积能把太阳能更多转换为电能，这两项只能减小稳定不能增加稳定性，增加支撑住的高度相当于提高了重心，会使结构更加不稳定。增大底座重量，降低了整个结构的重心，结构更稳定
