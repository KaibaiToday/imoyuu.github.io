第三章 制订我的设计方案

发现与明确设计问题

【第1题】
一个问题的解决或一个设计项目的进行，所受到的限制有（）。①设计对象的特点②设计标准的高低③设计者的技术能力④设计团队物力财力资源

A　①②④
B　①②③
C　②③④
D　①②③④
A B C D
答案：D
设计的限制主要有：①设计对象的特点②设计标准的高低③设计者的技术能力④设计团队物力财力资源。另外还有工期要求等。


【第2题】
某人一次无意中发现，碰翻的纺车锤竖立时仍能转动，由此得到启发发明了新式的纺纱机。这个案例中发现问题的途径和方法是（）

A　观察日常生活
B　由别人给出
C　收集和分析信息
D　技术研究和试验
A B C D
答案：A
某人一次无意中发现，碰翻的纺车锤竖立时仍能转动，这是对日常生活的观察。


【第3题】
下列哪一个问题具有研究的价值(　　)

A　能否制造没有任何电磁辐射的手机
B　能否把水变成汽油
C　能否制造一种不用能源的手表
D　能否制造利用太阳能的汽车
A B C D
答案：D
A、B、C均是违反科学规律的。


【第4题】
某设计师查阅相关数据后，发现火灾中大多数人死亡的原因是吸入毒性气体。于是他设计了如图所示的消防装置，由简易灭火器和防毒面具组成，既能灭火又能保护使用者。该设计师发现问题的途径是(　　)



A　观察日常生活
B　技术试验
C　技术研究
D　收集和分析信息
A B C D
答案：D
某设计师查阅相关数据后，这是通过对现有信息的收集和处理之后发现的问题。


【第5题】
某同学在旅行中，发现很多小学生会在炎热的暑假吃力地拖着大而重的行李箱到外地参加夏令营或者研学活动。她发明了一款能自动跟随的行李箱，她发现问题的途径是（　　）。

A　观察日常生活
B　收集和分析信息
C　设计与制作

A B C D

发现问题的途径一般有观察生活、 调查研究、技术试验。改同学在旅途生活中发现问题当然是源于对日常生活的观察。

【第8题】
学校外共享单车停放点场地小，许多同学又乱停乱放。晚来的同学经常无法停放，学校希望同学们提出合理的解决方案。小李同学设计解决办法时，受到了各种主客观条件的限制，以下不属于限制条件的是（）

A　小李的知识储备及实践能力
B　设计所需的时间及经费等
C　日常停放单车数量
D　共享单车的材质
A B C D
答案：D
设计的的限制主要有：个人或团队的设计能力；设计需要的材料、设备和技术；工期、经费、场地等。共享单车的材质不属于限制条件。

【第10题】
张明参考网上新产品杯托的图片，用1.5毫米厚的钢板制作了一个如图3-3所示(依据国家标准，技术图样上的尺寸，以毫米为单位时，不注写单位)的简易杯托，安装在书面的木板上，用来放置水杯。为了观察水杯在杯托中是否稳定，他找来各种不同直径的水杯和饮料瓶，分别放到杯托中，发现当水杯或饮料瓶的直径小到一定程度时，受到外力后容易晃动，甚至会从杯托中翻出来。张明发现问题的途径是()



A　收集信息
B　分析信息
C　观察日常生活
D　技术研究与技术试验
A B C D
答案：D
考点: 技术素养(工程思维)，学科内容(发现问题的途径)。解析:发现问题的途径:观察日常生活、收集和分析信息、技术研究与技术试验等。技术研究、技术试验是重要的技术活动，也. 是发现问题的途径与方法。通过技术研究、技术试验，我们有可能从对已有技术问题的研究中发现与之相联系的问题，从已有的研究结论中发现新的问题，也有可能在技术研究、技术试验过程中获得灵感、体悟，进而发现新问题。


【第11题】
问题的来源多种多样。当汽车的速度超过100米每小时时，空气阻力越来越明显。发现者把问题反馈给汽车生产公司，公司通过研发，设计出了外观呈流线型的汽车。就技术问题的来源而言，这是( )

A　人类生存必然会遇到的问题
B　由别人给出问题，设计者必须针对问题寻求解决方案
C　汽车生产公司人员结构的变化催生出来的问题
D　基于一定的目的，由设计者自己主动地发现问题并试图解决
A B C D
答案：B
考点:技术素养(技术意识)，学科内容(问题的来源)。解考点:技术素养(技术意识)，学科内容(问题的来源)。解由别人给出问题，设计者必须针对问题寻求解决方案;基于一定的目的，由设计者自己主动地发现并试图解决的问题。题目中是发现者把此问题反馈给汽车生产公司，公司通过研发，设计出了外观呈流线型的汽车。

【第14题】
通用技术老师上完课后，让大家回家后自己动手设计制作个简易的小台灯给幼儿园的小朋友使用。小明思考后，发现制作小台灯困难重重。他列举了以下问题： ①资金有限 ②课后作业较多，时间不够用 ③小台灯使用的电压要控制在36V安全电压以内 ④设计小台灯造型时要充分考虑幼儿园儿童的心理需求 ⑤自己动手能力较弱 请问以上问题中哪些是小明设计简易小台灯的限制因素()

A　①②③④⑤
B　①②
C　C①②⑤
D　①②③⑤
A B C D
答案：Ａ
解析:设计并不完全是自由的，设计的过程中，会遇到很多现实的问题，受到很多限制。需要我们认真考虑设计对象的特点，另外，还需要考虑设计人员自身的限制条件，以及成本、制造工艺、环境等其他限制条件。以上问题全部是限制因素。
