第二章 技术设计的基础

设计的创造性思维和工程思维

【第1题】
新款车载音箱设备使用非常便捷，只要与手机进行过一次蓝牙连接,以后每次上车， 音箱都会自动寻找和连接手机蓝牙并启动酷狗音乐等播放程序，整个过程不再需要人工干预。这种创新属于( )

A　结构创新
B　外观创新
C　人机交互方式创新
D　原理创新
A B C D
答案：C
解析：传统设备需要由人完成某种功能，创新后的设备可以改变了人机交互方式


【第2题】
如果要生产笔记本电脑，需要对资金、工厂、生产线甚至企业注册和商标注册等 多种要素进行分析，这样的思维方式属于( )

A　收敛思维
B　正向思维
C　发散思维
D　逆向思维
A B C D
答案：C
解析：从题干中得知要考虑多种要素，全方位的考虑问题，涉及面越宽越好，这是发散思维


【第3题】
科学家仿照水母的听觉系统，发明了“水母耳风暴预测仪”，此发明的主要运用的构思方法是（）

A　草图法
B　移植借鉴法
C　列举法
D　联想法
A B C D
答案：B
解析：仿照水母的听觉系统，发明了“水母耳风暴预测仪”运用的是模仿法。模仿法的核心在于通过别人的想法、构思，激发自己的灵感。


【第4题】
盲人用的"探路仪”是根据蝙蝠超声定位原理，内装一个超声波发射器，使盲人可以发现电杆、台阶、桥上的人等，这采用的方案构思法是( )

A　联想法
B　移植借鉴法
C　草图法
D　列举法
A B C D
答案：B
解析：模仿法的核心在于通过别人的想法、构思，激发自己的灵感。 模仿法在仿生学领域的应用:模仿鹰眼的成像原理仿制出来的微型照相机、摄像机自动调焦的摄像头等人造卫星的拍摄系统;模仿飞鸟的翅膀结构原理制造出的飞机机翼等。模仿是对原理拷贝。 模仿法构思设计方案时，可以大致分为: a.外形模仿 b.仿生模仿:形态仿生、结构仿生、原理仿生 c.结构模仿 d.功能模仿
