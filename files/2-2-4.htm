第二章 流程及其设计

流程的优化

【第1题】
昆明地铁是西南地区首个支持乘车码的地铁，使用乘车码乘坐地铁的流程大致为：进站刷码过闸→乘坐地铁→出站刷码出闸→自动扣费，“乘车码”为乘客提供多元支付体验的同时，更提升了轨道交通的服务水平，助力春城市民文明出行。结合此案例，从流程的角度分析，下列观点错误的是（ ）。

A　该流程可以免去排队充值、现金找零等方面的困扰
B　引入“移动支付”自动扣费，主要属于流程的技术优化
C　使用乘车码可享受“一分钱乘地铁”等优惠活动，属于成本优化
D　使用乘车码乘坐地铁“先乘车，后付费”，也属于流程优化
A B C D
答案：C
解析：流程优化的指标有工期优化、工艺优化、成本优化、技术优化、质量优化等。 C选项使用乘车码可享受“一分钱乘地铁”等优惠活动的目的是为了鼓励大家乘坐地铁，低碳出行，其目的是节能减排，保护环境与成本优化无关。其他选项表述正确。


【第2题】
在食品生产加工企业,工人进入生产车间前都要经过消杀、换装环节，主要原因是（ ）

A　节约生产时间
B　保证产品质量
C　改进生产工艺
D　降低生产成本
A B C D
答案：B
解析： 此流程优化是增加了消杀和换装环节,没有节约时间也不是改变生产工艺,还会增加成本,这些都是为了保证产品质量。


【第3题】
下列对流程优化的判断不正确的是()



A　教师批阅试卷时从手I（手工）批阅改为网上批阅属于I期（ 工期）优化
B　物流公司每天对各个投递员投递路线的优化属于质量优化
C　加工法兰时从切削法改为少切削法属于工艺优化
D　集成电路生产流水线上，机器人焊接代替手工焊接属于成本优化
A B C D
答案：B
解析： 本题考查流程设计应考虑的基本因素，流程优化及目的。流程设计要依据事物的内在属性和客观变化规律，有些流程的优化，增加工序是为了提高生产效率，新型设备的引入可以优化流程。教师批阅试卷时从手工批阅改为网上批阅属于工期优化，正确；加工法兰时从切削法改为少切削法属于工艺优化，正确；物流公司每天对各个投递员投递路线的优化属于路线优化，错误;集成电路生产流水线上，机器人焊接代替手工焊接属于成本优化，正确。
