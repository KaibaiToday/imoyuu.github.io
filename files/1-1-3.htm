第一章 技术及其巨大的作用

感受设计与技术的关系

【第1题】
下列关于技术与设计关系的描述，不正确的是( )

A　设计是技术发展的重要驱动力
B　技术的进步可以促进人们设计思维和手段的发展
C　技术与设计是相互依存、相互促进的
D　技术发展对设计不会产生影响
A B C D
答案：D
解析：设计是技术发展的重要驱动力，技术的进步可以促进人们设计思维和手段的发展，技术与设计是相互依存、相互促进的。


【第2题】
以前，电影中的“撞车”特技由特技人员完成，风险较大。随着技术的进步，如今改用机器人来完成，并结合电脑后期加工，使得特技效果更惊险、更逼真、更完美。这主要体现了（）

A　设计的创新促进了技术的发展
B　设计制约了技术的发展
C　技术具有丰富的内涵
D　技术的发展对设计起了重要影响
A B C D
答案：D
解析：设计与技术的关系，技术的发展为设计提供更大的可能。


【第3题】
麒麟990是华为研发的新一代手机处理器。海思麒麟990处理器将会使用台积电二代的7nm工艺制造。这个工艺加入后，可以让新的7nm制程，相比上代在性能提高20%的前提下，功耗却下降20%，麒麟990的另外一个重磅升级点，就是集成了5G基带，处理器中同时封装了AP(应用处理器)和BP(基带处理器).集成5G基带，让手机的续航提升。这个案例说明了(?)



A　技术的发展离不开设计
B　技术的发展超越设计
C　设计具有丰富的内涵
D　技术进步制约设计的发展
A B C D
答案：A
解析：此题考察技术与设计的关系。做题时，要读出技术和设计哪个出现在前，哪个出现在后，一般，先出现的促进后出现的；后出现的离不开先出现的。后出现的制约先出现的。二者是相互促进相互制约的关系。此题中，集成5G基带的设计促进了处理器升级技术的进步，提高了处理器的性能。
