第二章 技术设计的基础

技术设计的一般过程和方法

【第1题】
某校举办运动会，要求各班设计制作活动展板。设计制作过程的步骤包含：①制作、检测、评价和改进；②确定项目主题，制定设计目标；③提出解决问题的方案，权衡论证、决策；④详细的设计基本参数、功能、材料选取、人员安排等。其正确的顺序是（）。

A　③②①④
B　②③④①
C　③①②④
D　②①④③
A B C D
答案：B
解析：设计的一般过程是：确定问题；明确设计要求；制定方案；制作原型或模型；检测、评价和优化。


【第2题】
设计一款可挂在阳台栏杆上的小桌，则在设计该产品时，以下因素中不需要考虑的是（）。

A　小桌的制作方法
B　小桌的尺寸
C　栏杆的品牌
D　栏杆的尺寸
A B C D
答案：C
解析：栏杆的品牌不影响小桌的设计。


【第3题】
特殊人群是社会的重要组成部份，他们往往有着独特的需要。设计时，以下哪个设计不是针对特殊人群的特点和需要来进行设计（ ）。

A　盲道
B　平缓的坡道
C　地铁站内设置垂直电梯和坡道电梯
D　地铁车厢内的扶手
A B C D
答案：D
解析：地铁车厢内的扶手是所有人都需要用的。


【第4题】
在发现与明确问题的基础上，紧接着制定设计方案的顺序为(　　)。

A　收集信息、方案构思、设计分析、方案呈现、方案筛选
B　收集信息、方案构思、方案筛选、设计分析、方案呈现
C　设计分析、收集信息、方案构思、方案筛选、方案呈现
D　收集信息、设计分析、方案构思、方案呈现、方案筛选
A B C D
答案：D
解析：制定设计方案的流程是收集信息、设计分析、方案构思、方案呈现、方案筛选。


【第5题】
根据我国《道路交通安全法》的规定，驾驶或乘坐摩托车不戴头盔都是交通违法行为，都会受到交通处罚。驾驶或乘坐摩托车戴头盔主要是为了实现人机关系中的（）目标。





A　安全
B　舒适
C　高效
D　健康
A B C D
答案：A
解析：佩戴头盔保证安全
