技术试验

【第1题】
我国新一代载人飞船试验船底部安装6个气囊帮助舱体平稳软着陆。为了保证气囊在发射时处于稳定可靠的折叠包装状态，在需要充气展开工作时又能够迅速顺畅打开，设计团队前期进行了大量地面充气展开试验。这种技术试验方法是 ( )

A　模拟试验法
B　移植试验法
C　对比试验法
D　计算机仿真试验法
A B C D
答案：Ａ
〖优选试验〗：用于设计参数或设计方案的对比和筛选〖模拟试验〗：通过人为设计试验场景来模拟真实发生情况的试验，〖强化试验〗：产品在出厂前，为了提前获取产品的使用寿命数据，人为地强化试验条件，〖虚拟试验〗借助计算机软件进行的虚拟仿真试验。


【第2题】
为确保国旗在1949年10月1日开国大典中分秒不差地升起，技术小组克服重重困难，严格按比例缩小了场景，模拟了国旗升降过程，发现并及时解决了问题。以下关于技术试验的说法不正确的是( )。

A　在技术设计活动中，技术试验是必不可少的
B　通过技术试验，可以降低设计的风险和失误概率
C　通过技术试验，可以发现问题、推动技术设计的发展
D　技术试验有很多种，本案例中采用的是移植试验法
A B C D
答案：Ｄ
在技术设计活动中，技术试验是必不可少的；通过技术试验，可以降低设计的风险和失误概率；通过技术试验，可以发现问题、推动技术设计的发展。〖对比试验〗：用于设计参数或设计方案的对比和筛选〖模拟试验〗：通过人为设计试验场景来模拟真实发生情况的试验，〖强化试验〗：产品在出厂前，为了提前获取产品的使用寿命数据，人为地强化试验条件，〖计算机仿真试验〗借助计算机软件进行的虚拟仿真试验。〖移植实验〗：在相互具有差异的事物之间，将某些共同相关的因素从一物移植到另一物进行的实验。在相互具有差异的事物之间，将某些共同相关的因素从一物移植到另一物进行试验。本案例是模拟试验。


【第3题】
如图所示是某企业研发的小型金属探测器。当探测到金属物时，蜂鸣器会发出报警声。以下对该产品性能测试选项中不需要的是(　　)。



A　在潮湿的环境下测试其工作的稳定性
B　用硬物敲击金属探测器表面测试其抗击能力
C　在不同的金属物周围探扫，进行灵敏度测试
D　探测金属物，测试蜂鸣器是否能正常发出报警声
A B C D
答案：B
该产品的性能主要是探测金属，所以在其性能测试中不需要测试其表面的抗击能力
