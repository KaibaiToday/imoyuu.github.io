第三章 制订我的设计方案

技术设计方案

【第1题】
设计学生用的台灯时要考虑的因素很多，归结起来分别属于“物”、“人”、“环境”三个方面。以下设计分析中从“物”的角度分析的是(　　)

A　学生看书的时间较长，光线要柔和
B　根据学生的特点，外形要新颖美观
C　尽可能采用标准件，简化制作
D　应适合在书桌、床头等场合使用
A B C D
答案：Ｃ
“物”指产品本身，包括产品的功能、结构、材料等，采用标准件制作属于“物”的因素。


【第2题】
小李参考如图所示的产品，准备用木条制作一款可折叠的饮水桶支架。他提出的结构方案中合理的是（）

A　选Ａ
B　选Ｂ
C　选Ｃ
D　选Ｄ
A B C D
答案：A
选项D的结构在使用时不稳定；选项B的结构不方便折叠；选项C右侧的插销强度不足，会导致在使用时无法支撑饮水桶的重量而变形；选项A设计的结构既可以折叠，又能够保障结构的稳定性。


【第3题】
小王准备设计一款电动吸尘黑板擦。设计时不需要考虑的是（）。



A　教室讲台高度
B　黑板擦的供电方式
C　黑板擦的结构
D　黑板擦的吸尘方式
A B C D
答案：Ａ
教室讲台高度不属于设计电动吸尘黑板擦的考虑因素。


【第4题】
如图所示是一款双把手弯刀，它能方便迅速切一些质地坚硬的蔬菜或水果，降低划伤手指的可能。而可折叠的把手可防止锋利的刀片生锈以及造成意外伤害。设计该产品时，以下因素中从“环境”的角度考虑的是（）



A　刀刃的厚度
B　常人的握力
C　蔬菜或水果的硬度
D　刀柄的大小
A B C D
答案：C
刀刃和刀柄属于“物”的因素，常人的握力是“人”的因素，只有蔬菜和水果的硬度属于环境因素。


【第5题】
健康饮水是生活中一个不可忽视的问题。为了在户外能方便、健康的饮水，随身携带水壶成为大多数人的选择。某学生想要设计一个中学生随身携带的水壶，他对水壶进行了设计分析。对产品进行设计分析应考虑的因素可归结为三个方面，它们是（）



A　物、人、环境
B　美观、实用、经济
C　结构、功能、安全性
D　高效、健康、舒适
A B C D
答案：Ａ
设计分析的三要素是人、物、环境。


【第6题】
两桌脚间的连接设计，由方案1改进到方案2，主要是为了( )



A　便于安装时定位
B　提高连接的可靠性
C　减少制作材料
D　提高桌脚的强度
A B C D
答案：B
解析：梯形截面形状相对矩形截面形状提高了连接的可靠性，能够更好地避免结构连接处的转动或者偏移。


【第7题】
李老师家的书房很狭小，为了设计一款合适的书桌，李老师请多个设计师设计了几个不同的方案，对比后选取了一个对空间利用更合理的方案，这个阶段是制定设计方案中的（　　）



A　方案构思
B　方案呈现
C　方案优化
D　方案筛选
A B C D
答案：Ｄ
从多个方案中，通过分析、比较和权衡，选择一个更合适的方案，属于方案筛选。


【第8题】
某同学对台灯的三个设计方案进行比较评估,列表如下。下列选项中说法正确的是( )。



A　方案1的成本最低
B　方案1的美观程度优于方案3
C　方案2比方案1更优
D　三个方案的结构稳定程度一样
A B C D
答案：C
从5个指标综合考虑，方案2比方案1好。


【第9题】
( )用于分析、构思和推敲,有快速记录创意的作用。



A　透视图
B　构思性设计草图
C　三视图
D　表现性设计效果图
A B C D
答案：Ｂ
解析:设计草图分为构思性设计草图和表现性设计效果图。构思性设计草图用于分析、构思和推敲，有快速记录创意的作用。表现性设计效果图用于最终设计方案的呈现，需要将构思好的形态、色彩、材质准确地绘制出来，用于设计交流。


【第10题】
现在，许多技术产品的体积越来越小,耗能越来越小,携带越来越方便，使用功能越来越完善. ... .这在很大程度上都是( )带来的。

A　人机交互方式创新
B　结构创新
C　原理创新
D　外观创新
A B C D
答案：B答案选项间有交叉，尤其结构创新与原理创新
解析:创新设计的类型有原理创新、结构创新、外观创新、人机交互方式创新。题目中应该是结构创新。


【第11题】
新学期伊始，，老师安排小光设计制作个运动会的引导牌。小光在网上査了大量图片资料，分析研究后用长600mm、宽400mm的木板和直径35mm、长1000m的木混做了个引导牌(如图所示)。使用时小明发现引导牌不便于携带，准备重新设计制作一个可以收和展开的引导牌。构思引导牌的结构方案时不需要考虑的是（）



A　展开后如何保持结构稳定
B　手柄和面板如何收拢和展开
C　面板的颜色
D　手柄与面板如何连接

答案：C
解析：颜色不是该设计的限制条件，不需要考虑。
