第四章 控制及其设计

控制中的干扰

【第1题】
家用普通鱼缸通过水温控制系统可使鱼缸水温保持在设定的温度。其控制方框图如下，下列属于鱼缸温度控制系统干扰因素的是（）



A　电压波动
B　更换大功率加热棒
C　从鱼缸中放出少量水
D　向鱼缸加入凉水
A B C D
答案：D
向鱼缸加入凉水将会引起鱼缸内水温的变化，属于干扰因素。


【第2题】
如果将人骑自行车前进看作一个控制系统的话，下列不属于干扰因素的是（）

A　上坡
B　逆风
C　主动变更路线
D　路滑
A B C D
答案：C
本题考察闭环控制系统的干扰因素。 由题干分析人骑自行车是前进是一个不断调整的过程，所以这个控制系统是闭环控制系统。 干扰指的是对控制系统的行为造成影响的有害的信号及扰动。即除给定量外，引起被控量量值变化的各种因素。 本题中上坡、逆风、路滑是都是外界因素干扰人骑自行车前进属于干扰因素；而主动变更路线是人为的也可以认为是系统的输入量产生了变化而改变了输出量不属于干扰因素。故选：C。


【第3题】
如图所示是自动门示意图。当人走进感应区域时，检测装置探测到人体信号，控制电路接收到该信号后，发出相应的指令，启动电动机把门打开。当门开启之后，控制电路将作出判断，如果没有人在感应区域内，就通知电机做反向运转，将门关闭。关于该控制系统，下列说法中正确的是（）



A　输入量是人体信号，所以是人工控制
B　门开启之后可以自动关闭，存在反馈环节
C　自动门的开启或关闭与检测装置是否检测到人体信号相对应
D　有人在感应区内从门口经过不会对系统的控制产生干扰
A B C D
答案：C
人体信号只是输入量，没有全程干预，属于自动控制，A错误；输出量门开启或关闭对系统的控制没有影响属于开环控制，不存在反馈环节，B错误；检测装置检测到人体信号时，自动门开启，没有检测到人体信号时，自动门关闭，C正确；有人从门口经过，不需要门打开，对输出量产生了影响，又不是输入量，会对系统的控制产生干扰，D错误


【第4题】
冰箱的温度控制系统中，不属于影响其温度控制的干扰因素是( )

A　频繁开关冰箱
B　把热菜放入电冰箱中
C　电冰箱所在房间的温度从20℃变到30°C
D　把电冰箱冷藏温度设定为4℃
A B C D
答案：CD
解答本题关键在两处，一处是知识储备，即明确何为控制系统的干扰因素:在控制系统中，除输入量以外，引起被控量变化的各种因素称为干扰因素。另一处是审题，题中问的是不属于干扰因素的是除了“把电冰箱冷藏温度设定为4℃”外的其他三个选项都会引起冰箱内温度变化，属于干扰因素把冰箱冷藏温度设定为4℃属于输入量引起的温度变化，不属于干扰因素。


【第5题】
可以定时的台灯可以节省用电,将拉绳拉到设定时间位置,打开灯的同时定时器开始工作,到达设定时间开关断开,将灯熄灭。关于这款台灯的控制系统,下列说法正确的是()

A　控制器是拉绳
B　输入量是设定的时间
C　控制量是台灯的亮或灭
D　执行器是台灯
A B C D
答案：B
执行器是开关;控制器是电路,控制量是台灯的关与开


【第6题】
监控摄像系统被广泛用于记录道路交通实时图像信息。对于监控摄像系统,下列环境因素中不属于干扰因素的是（ ）

A　沙尘
B　噪声
C　闪电
D　浓雾
A B C D
答案：B
在控制系统中,除输入量(给定值)以外,引起被控量变化的各种因素都可以视为干扰因素。被控量即输出量即监控摄像头拍摄到的实时图像信息,噪声并不会引起图信息的变化,所以,噪声不属于干扰因素。


【第7题】
2022年北京冬奥会比賽场馆的各种控制系统会遇到各种干扰因素,在硏究扰对控制系统的影响时,主要是观察:()

A　对控制系统组成产生的影响
B　对输入量产生的影响
C　对控制系统控制量产生的影响
D　对控制系统的输出量产生的影响
A B C D
答案：D
在控制系统中,除输入量(给定值)以外,引起被控量变化的各种因素都可以视为干扰因素。被控量=输出量。


【第8题】
家用电冰箱温度控制中不属于干扰因素的是()

A　把食物放到冰箱内
B　房间温度变化很大
C　市电电压下降
D　打开冰箱门
A B C D
答案：C
在控制系统中,除输入量(给定值)以外,引起被控量变化的各种因素都可以视为干扰因素。被控量=输出量。市电电压下降属于故障,不属于干扰因素。


【第9题】
25、空调使人们的生活更舒适。在室内空调温度控制系统中，以下属干影响其温度控制的干扰因素是（ ）

A　室内人数增加
B　室内窗户被打开
C　在室内烧水
D　以上都是
A B C D
答案：D
本题考察的是控制系统的干扰因素，一般我们把对一个系统的运行，会带来影响的因素，叫做干扰因素。干扰作用一般表示在被控对象上。每个人都有一定的体温，室内人数增加，室内热量肯定相应增加，属于干扰因素。室内窗户被打开，和室外有热量交换，会形成干扰因素。在室内烧水，会使室内温度升高，属于干扰因素。
