【第1题】
小明用微波炉加热食物，设定时间为一定时间，时间到后，微波炉停止工作。这个控制系统属于（）

A　开环控制、人工控制
B　闭环控制、人工控制
C　闭环控制、自动控制
D　开环控制、自动控制
A B C D
答案：D
开环控制系统的特点是，输入信号到输出信号之间的信息传递是单向的，没有反馈通道，所以这是一个开环控制系统。又因系统开始工作后不再借助于人工千预，所以是自动控制


【第2题】
电饭锅在使用中存在着“煮饭"和“保温”两个过程(即两个控制系统) 。关于“保温”控制系统,下列判断正确的是（ ）

A　人工开环控制系统
B　自动开环控制系统
C　人工闭环控制系统
D　自动闭环控制系统
A B C D
答案：D
电饭锅的“保温”控制，在对温度进行控制时没有人全程参与，故属于自动控制。又由于该控制系统输出是实际温度，输入时温度的设定值，可以形成反馈，故属于闭环控制系统。综上所述，本题答案为：D


【第3题】
以下对饮水机水温控制系统的类型和方式描述正确的是（）

A　闭环手动控制系统
B　闭环自动控制系统
C　开环手动控制系统
D　开环自动控制系统
A B C D
答案：B
其中开环和闭环的区别在于有没有反馈，手动和自动的区别在于人在控制过程中是否直接起到控制作用。 饮水机水温控制系统无人直接参与的情况下烧水，并将水温保持在设定值，故是自动控制；另外温度控制系统中温度传感器作为反馈装置将温度反馈回控制电路，故属于闭环控制系统。故选：B。

【第9题】
王刚同学放假在家,用微波炉热菜汤,设定加热时间为3分钟。按加热按钮3分钟后,微波炆停止加热。这种微波炉的加热控制系统为()

A　闭环自动控制系统
B　开环手动控制系统
C　开环自动控制系统
D　闭环手动控制系统
A B C D
答案：C
微波炉的停止加热是由时间决定的,而不是由控制系统的输出量“菜汤的温度”决定的,即便菜汤放凉,微波炉也不会再次启动工作。所以,该控制过程的输出量并不影响控制过程,并不存在反馈环节,是开环控制系统。

【第12题】
控制系统工作的过程,实质是信息的流动过程。下列关于控制系统工作过程的说法,正确的是（）

A　开环控制系统的信息只能从输入端传至输出端
B　闭环控制系统的信息只能从输出端传至输入端
C　信息只能从输入端传至输出端
D　信息只能从输出端传至输入端
A B C D
答案：A
因为闭环控制系统的信息传输是一个完整的回路,所以,总的来说信息既可以从输入端传至输岀端,也可以从输岀端传至输λ端。但是,开环控制系统的信息传输是单向的,没有反馈环节,所以,只能是从输入端传至输出端。


【第13题】
学习控制,就要比较闭环控制系统和开环控制系统,下列判断错误的是()

A　闭环控制系统和开环控制系统都有比较器
B　开环控制系统的控制精度较低
C　开环控制系统无反馈环节
D　闭环控制系统的抗干扰能力强
A B C D
答案：A
根据控制系统的输出量是否对控制产生影响,可分为开环控制系统和闭环控制系统。开环控制系统的输岀量不对系统的控制产生仼何影响。闭环系统的输岀量返回到输岀端,并对控制过程产生影响。在闭环控制系统中,将输岀量通过检测装置返回到输入端并与输入量进行比较的过程,就是反馈。开环控制系统没有反馈环节。闭环控制系统就是通过反馈环节克服干扰因素,从而使系统的被控量稳定在给定的范围内,所以对于—个控制问题,闭环控制系统的控制精度通常比开环控制系统髙,抗干扰能力强。

【第19题】
公共场所都要安装火灾报警系统。当烟雾探测器检测到烟雾浓度达到设定值时,控制器控制报警器发出声光信号进行报警。该控制系统属于( )

A　人工控制、开环控制
B　自动控制、闭环控制
C　人工控制、闭环控制
D　自动控制、开环控制
A B C D
答案：D
控制按控制的方式分为: 1.人工控制:由人本身通过判断和操作进行控制，需要人的直接千预和全程干预。2.自动控制:不需要人工操作，使机器、设备自动地按照预定的规律工作。 按基本组成与工作过程分为: 1.开环控制系统:控制系统的输出量不对系统的控制产生任何影响。2.闭环控制系统:控制系统可以把输出量返回到输入端与给定值进行比较，构成一条闭合回路，即系统的输出量返回到输入端并对控制过程产生影响。 火灾报警系统属于自动触发，没有反馈环节的一个系统，所以是自动控制、开环控制。
