第三章 系统及其设计

系统的设计

【第1题】
家用轿车都有倒车雷达。倒车雷达的工作过程是:车尾的探头传感器接收超声波信号传递给控制器，控制器依据车体与障碍物间的距离和方位信息，控制蜂鸣器是否发岀警报。为提高该系统的性能，需对其进行优化，以下不属于影响该系统优化的因素是( )

A　探头的探测精度
B　障碍物和车之间的距离
C　探头的数量
D　控制器的性能
A B C D
答案：B
在系统优化中，影响系统优化的因素是指对系统的目标函数(优化目标)产 生显著影响，并且可以人为调节的因素。属内因。在该题目中，探头的数量、控制 器的性能和探头的探测精度都是属于倒车雷达控制系统内部可调控的要素，属內 因。而障碍物和车之间的距离属于系统外部因素。因此，该题答案选择探头的数 量、控制器的性能和探头的探测精度。


【第2题】
高架桥大多设计有雨水回收系统，可以将回收的雨水处理后，用于浇灌树木、草坪，从而节约了水资源。关于该系统的设计不需要考虑的是（）

A　雨水回收系统建设成本
B　高架桥的结构
C　高架桥车流量的变化
D　桥面积水的问题
A B C D
答案：C
该题考查的知识点是“系统设计应考虑的主要问题”。“桥面积水的问题、雨水回收系统建设成本、高架桥的结构”这三个系统要素之间相互联系、相互作用，是该系统设计要考虑的主要因素。“高架桥车流量的变化”对该系统的设计影响不大。


【第3题】
谷歌自动驾驶汽车依靠人工智能、视觉计算、雷达、监控装置和全球定位系统协同合作，让电脑可以在没有任何人类主动的操作下，自动安全地操作机动车辆。以下关于该系统的说法中，正确的是( )



A　设计时既要考虑乘坐的舒适性、行驶的安全性，还要考虑设计制造成本、市场应用前景等，体现了系统分析
B　多种传感器配合，能应对各种天气的变化及复杂的路面状态，体现了系统的环境适应性
C　视觉计算能准确判断红绿灯及计时状态，体现了系统的动态性
D　该系统中某个传感器出现故障，导致采集的相关信息出错，体现了系统的整体性
A B C D
答案：B
本题考查系统分析的原则和系统的特性。 多种传感器配合，能应对各种天气的变化及复杂的路面状态，体现了系统的环境适应性，正确;视觉计算能准确判断红绿灯及计时状态，体现了系统的目的性，错误;该系统中某个传感器出现故障，导致采集的相关信息出错，体现了系统的相关性，错误;设计要综合考虑舒适性、安全性、制造成本和市场应用前景等目标，体现系统分析的综合性原则，错误。


【第4题】
即时回放系统在球类比赛中已被广泛应用(如图所示为网球)，当球员对裁判的判罚有争议时可以提出W挑战。”该系统由多个高速摄像头、电脑和大屏幕组成。首先通过电脑的计算把比赛场地内的立体空间分隔成毫米级的测量单位;然后利用高速摄像头从不同角度同时捕捉网球飞行轨迹的基本数据;再通过电脑计算，将这些数据生成3D图像;最后利用肌时成像技术，由大屏幕清晰呈现出网球的运动路线及落点。下列对该系统分析不正确的是( )



A　系统能在10秒内完成数据的采集和结果的演示，体现了系统的目的性
B　为了优化摄像头安装位置，网球场地的大小是影响系统优化的因素
C　高速摄像系统和3D即时成像系统是即时回放系统的两个子系统
D　通过定量分析，确定需设置8台高速摄像头，体现了系统分析的科学性原则
A B C D
答案：B
本题考查系统与设计，系统的目的性，系统分析的相关知识点。系统的优化是指再给定的条件下，根据系统的优化目标，采用一定的手段和方法，使系统的目标值达到最优。影响系统优化的因素是指能对系统的目标产生显著影响，并且能够人为调节的，网球场的大小是规定尺寸，不是人为可以调节的因素，故“为优化摄像头安装位置，网球场地的大小是影响系统优化的因素”错误，符合题意。


【第5题】
如图所示为某公司新研发的一款智能垃圾桶，能够实现桶盖自动开合的功能。当人靠近智能垃圾桶时，红外传感器会将检测到的人体信号传递给单片机，单片机控制电机转动，使桶盖打开。从系统的角度分析，下列对智能垃圾桶的分析不正确的是（ )



A　在设计该智能垃圾桶时，先考虑自动开合功能的实现，再考虑制造成本，体现系统分析的整体性原则
B　智能垃圾桶需要定期更换电池，体现系统的动态性
C　设计该系统时，通过定量计算确定机器人所需的电池容量，体现了系统分析的科学性原则
D　单片机损坏，会导致智能垃圾桶无法自动开合，体现系统的整体性
A B C D
答案：A
在设计该智能垃圾桶时，先考虑自动开合功能的实现，再考虑制造成本，体现系统分析的综合性原则。系统分析的科学性原则是指系统分析一方面要有严格的工作步骤，另一方面应尽可能地运用科学方法和数学工具进行定量分析，使决策的过程和结果更具说服力。系统分析的综合性原则是指当系统存在若干个目标时，应将目标排出优先次序，首先实现最优先的目标，然后尽可能在不损害第一个目标的前提下完成下一个目标。这就是需要综合分析，统筹兼顾，不可顾此失彼，因小失大。
