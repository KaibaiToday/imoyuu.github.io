第二章 技术设计的基础

技术设计的原则

【第1题】
某同学设计制作了一款自行车防盗锁参加青少年科技创新大赛，该防盗锁能在小偷用钳子夹断锁扣时，瞬时释放高压电击小偷，阻止偷窃行为。作品在初审就被否定，其原因最有可能是（）

A　作品不够创新
B　作品不符合道德原则
C　作品不够实用
D　作品不够美观
A B C D
答案：B
解析：该设计虽然有一定的技术含量，但高压容易造成人身伤害，不符合法律、道德规范原则。


【第2题】
书包背带增宽的设计主要是为了实现人机关系目标中的( )

A　安全
B　高效
C　舒适
D　经济
A B C D
答案：C
解析：背带增宽，背者背包的感觉更舒适。


【第3题】
人行天桥的设计中，通常都要在阶梯两边安排一定宽度的斜面通道。这是为了（）

A　美观
B　满足不同人群的需要
C　提高强度
D　节省材料
A B C D
答案：B
解析：充分考虑各类人群的需要。


【第4题】
我国家用电器使用的交流电一般额定电压是220V、频率是50Hz，这说明家用电器的设计符合（）

A　科学性原则
B　实用性原则
C　标准化原则
D　可持续发展原则
A B C D
答案：C
解析：我国家用电器使用的交流电一般额定电压是220V、频率是50Hz，统一规范，符合标准化原则。


【第5题】
电脑主机后面的各种插口都是通用的，这是由于设计时遵循了( )。



A　实用性原则
B　经济性原则
C　标准化原则（技术规范化原则改为标准化原则）
D　可持续发展原则
A B C D
答案：C
解析：各种插孔通用，符合了技术规范性原则，方便维修和更换，使技术设备得以活用。
