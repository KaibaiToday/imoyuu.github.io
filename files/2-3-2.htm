第三章 系统及其设计

系统分析

【第1题】
我国有悠久的历史，也有数不清的历史典故。以下历史典故中主要体现了系统分析思想的是（）

A　田忌赛马
B　刻舟求剑
C　掩耳盗铃
D　守株待免
A B C D
答案：A
齐王和田忌的赛马方案共有6种，而结局有以3：0赢的，也有以2：1赢的，但只有一种情况是田忌以2：1取胜齐王的，孙膑通过排列组合找出了这种方案，充分体现了系统分析的科学性原则。


【第2题】
2020年7月31日上午10时30分，北斗三号全球卫星导航系统建成暨开通仪式在人民大会堂举行。中共中央总书记、国家主席、中央军委主席习近平出席仪式，宣布北斗三号全球卫星导航系统正式开通。从系统的角度分析，下列说法中不正确的是（ ）

A　北斗三号全球卫星导航系统能全天候向全球提供服务，体现了系统的环境适应性
B　导航系统的主控站、注入站、监测站和空间卫星相互影响，体现了系统的相关性
C　导航系统的主控站、注入站、监测站和空间卫星等共同构成了导航系统，体现了系统的环境适应性
D　导航系统具有定位、导航、授时、短报文通信等功能，体现了系统的目的性
A B C D
答案：C
C选项导航系统的主控站、注入站、监测站和空间卫星等共同构成了导航系统，体现的是部分组合形成了整体的系统功能，属于整体性，描述错误。 A、B、D选项均描述正确。?故选：C。


【第3题】
如图所示是农田智能灌溉系统示意图。系统根据监测的气象、土壤、农作物等相关信息，控制灌溉设备工作，使土壤保持合适的温度和吧力，达到降低人力成本、提高生产效率、节水节肥的目的。下列关于该系统的说法不正确的是（ )



A　实现了农田的智能化管理，体现了系统的目的性
B　灌溉设备需要定期维护，体现了系统的动态性
C　数据库能存储监测到的各种数据信息，体现了系统的环境适应性
D　数据库、信息采集等出现异常，会影响农田灌溉，体现了系统的整体性
A B C D
答案：C
本题考查系统及系统分析，系统各特性之间是相互联系的，掌握侧重点即可。系统的环境适应性是指系统能适应环境的变化。农田灌溉系统实现了农田的智能化管理，实现了功能，体现了系统的目的性，正确;数据库存储检测的信息并不能体现该系统的环境适应性，错误，但符合题意;系统处于不断变化发展中，设备需要定期维护，体现了系统的动态性，正确;数据库等出现异常，会影响农田灌溉，部分影响整体，体现了系统的整体性，正确。


【第4题】
如图所示是一款带显示屏的智能花盆，内置多种传感器和微处理芯片。用户手机与花盆间通过APP联网后，可手动设置基本参数等初始信息。花盆测得士壤的水分、光照和温度等，与设置好的参数进行比较，即可在屏幕上显示对应的动画，使花盆更加"人性化"”。用户可通过APP监护，使植物保持活力。从系统角度考虑，下列分析不正确的是（ ）



A　花盆由联网、微处理系统及显示系统等组成
B　任何一个部分损坏，都会影响智能花盆的工作，体现了系统的整体性
C　显示屏能显示动画，体现了系统的动态性
D　设计时兼顾”人性化"与成本，体现了系统分析的综合性原则
A B C D
答案：C
系统都是动态的，处在运动变化和发展之中。有“动画”并不是动态的表现，有动画不是必须必要的功能，而是更加人性化的设计。


【第5题】
如图所示的弹簧秤利用了弹性金属片的弹性进行称重。在秤盘上放置重物后秤盘位置降低，通过重量刻度尺可读出重物的重量。从系统的角度分析，下列说法不正确的是( )



A　长期使用后，弹性金属片的弹性会变差，体现了系统的动态性
B　弹性金属片的弹性系数和重量刻度尺的刻度相互关联，体现了系统的相关性
C　弹簧秤称量有一定范围，体现了系统的环境适应性
D　弹性金属片、重量刻度尺、秤盘和底座共同构成了弹簧秤系统，体现了系统的整体性
A B C D
答案：C
本题考查的是系统分析。对弹簧秤进行分析可知弹性金属片、重量刻度尺、秤盘和底座共同构成了弹簧秤系统，体现了系统的整体性。利用弹性金属片的弹性，秤盘受重时会下沉，体现了系统的相关性．长期使用后，弹性金属片的弹性会变差，体现了系统的动态性。弹簧秤的称量范围与采用的弹性金属片性能相关，体现的是整体性。与环境无关。不能体现系统的环境适应性，故错误。


【第6题】
如图所示的全自动面条机，将面粉、调料、水等原材料按一定比例放入后，面条机能自动和面制成面条面条机由控制面板、控制器主机、电机、搅拌器、模头等组成。以下关于该系统的说法中不正确的是( )



A　能自动和面制成面条，体现了系统的目的性
B　设计时先考虑电机，再考虑搅拌器，体现了系统分析的整体性原则
C　设计时既要考虑机器成本低，又要考虑人机关系好，体现了系统分析的综合性原则
D　电机各项参数要兼顾其他各部件，体现了系统的相关性
A B C D
答案：B
系统分析的整体性原则是指先分析整体再分析部分，先看全局再看局部等。“设计时先考虑电机，再考虑搅拌器，”是从局部到局部


【第7题】
如图所示的深海半潜式智能养殖场，安装各类传感器，可实现全自动监测、喂养、清洁、捕捞等功能。下列关于该智能养殖场系统的说法中正确的是( )



A　养殖场上浮或者下潜幅度的调整范围是增加养殖产量的影响因素
B　融入生物学、工学、电学、计算机等多学科技术，体现了系统分析的综合性原则
C　当检测到饲料浓度减少到下限时，投料系统会自动投放饲料，体现了系统的环境适应性
D　当清洁系统发生故障时，喂养系统将会受到影响，体现了系统的相关性
A B C D
答案：D
融入生物学、工学、电学、计算机等多学科技术，体现了技术的综合性，而不是“系统分析的综合性原则”。描述系统的功能不涉及系统的环境适应性，所以‘当检测到饲料浓度减少到下限时，投料系统会自动投放饲料，体现了系统的环境适应性”是不对的。清洁系统与喂养系统是该系统的两个子系统，两者相互影响体现了系统的相关性。养殖场上浮或者下潜幅度的调整范围，无法判断是否属于影响系统优化的因素.
