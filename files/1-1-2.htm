第一章 技术及其巨大的作用

认识技术与人、社会、自然的关系

【第1题】
大国工程发展迅速。中国铁路总公司牵头组织研制的”复兴”号动车组列车,具有完全自主知识产权、达到世界先进水平的动车组列车。在350千米时速下"复兴”号与"和谐”号相比，总能耗下降了10%。还增设碰撞吸能装置，以提高动车组被动防护能力。下列关于”复兴”号动车组列车的说法中不正确的是(?)

A　由中国铁路总公司牵头组织研制具有完全自主知识产权,体现了技术专利性
B　达到世界先进水平,体现了技术的创新性
C　增设碰撞吸能装置,以提高动车组被动防护能力,保护乘客的生命安全,体现了技术具有解放人的作用
D　在350千米时速下"复兴”号与"和谐”相比总能耗下降了10%，体现了可持续发展理念
A B C D
答案：C
解析：保护生命安全，体现了技术具有保护人的作用。


【第2题】
2020年是不平凡的一年。新冠肺炎疫情防控期间，京东在武汉启用了自行研制的智能物流小车,用于向武汉市第力人民医院配送医疗物资。这款物流小车实现了自主无人驾驶,有效减少了人员接触和感染风险。从中主要体现技术具有的作用是(?)。

A　提高人
B　解放人
C　发展人
D　保护人
A B C D
答案：D
解析：有效减少了人员接触和感染风险，保护人体安全。


【第3题】
宁安(南京南——马鞍山东——安庆)高铁的开通，给当地市民的出行，带来了很大的便捷，既节省了时间,人也不用那么辛苦。这说明技术具有（ ）



A　限制人的作用
B　解放人的作用
C　保护认定作用
D　发展人的作用
A B C D
答案：B
解析：技术的作用有保护人、解放人、发展人。交通工具解放了人的双腿，可以实现快速位移的转移。
