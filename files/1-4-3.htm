测试与优化

【第1题】
在不同设计阶段的模型中，与最终产品最接近的是（）

A　概念模型
B　结构模型
C　功能模型
D　展示模型
A B C D
答案：D
展示模型通常是在设计方案确定后，按所确定的形.态、尺寸、材质及表面效果等要求精细制作而成，外观与产品有相似的视觉效果，具有展示、宣传、设计交流与评价等作用。


【第2题】
某同学设计了一种安全门卡,可卡在普通房门上,以防止关门时意外夹手。若要测试门卡的性能,下列不需要测试的是( )

A　门卡材料的硬度
B　门卡的防水性
C　门卡与门板连接的牢固程度
D　门卡的尺寸
A B C D
答案：B
门卡卡放在门的边缘,高度应选择在门边缘的中间左右,离地面较高几乎不用考虑防水;门卡的尺寸要适合所卡门板的厚度否则会出现卡不进去或者因为太宽松卡不住,最好门卡张口尺寸有定的可变适用范围;必须考虑门卡所选材料种类及硬度,太硬了容易把门框磕坏,并且会有噪声,太软起不到防夹手的作用,必须选择软硬度合适的材料;门卡与门的连接既要牢固,还得考虑比较容易取下


【第3题】
李强同学通过通用技术的学习,提出了一些保温杯性能实验方案，下列试验方案中不合理的是（ ）

A　A渗漏试验:往杯中加水后,旋紧杯盖,把保温杯横放在桌面上,观察是否有水渗漏
B　B内胆异味试验;旋紧杯盖一段时间后打开杯盖,用鼻子感受内胆的气味
C　C跌落试验:用铁锤敲击保温杯,观察保温杯是否损坏
D　D保温试验;将开水倒入保温杯内,测量温度,旋紧杯盖,2小时后再测量水的温度
A B C D
答案：C
解析：“跌落试验”：用铁锤敲击保温杯， 观察保温杯是否损坏”前后矛盾，跌落试验与铁锤敲击矛盾。


【第4题】
如图所示是某位同学设计的一款木质台灯，现在要对其进行技术试验，下列试验中不需要的是（ ）







A　A调节灯杆, 测试其稳定性
B　B打开开关, 观察灯罩是否耐热
C　C打开开关, 测试电路是否正常
D　D打开开关, 测试灯泡的使用寿命。
A B C D
答案：D
解析：灯泡的使用寿命是灯泡这个产品的性能，在灯泡出厂前需要进行技术试验。不属于台灯的性能试验范畴。
