【第1题】
小王在生活中发现倒酱油、倒醋不方便，于是设计了一款新型醋瓶，如图所示为新型醋瓶及其评价坐标图。从坐标图可知该醋瓶（）

B　不美观

D　价格昂贵

【第2题】
如图所示是一款新型课桌椅，客户试用后做出如下评价：设计新颖美观；桌椅连体结构和下部储物能充分利用空间；移动方便，适合分组学习。以上评价属于(　　)





A　对最终产品的评价
B　对设计过程的评价
C　对设计方案的评价
D　自我评价
A B C D
答案：A
设计的评价是指依据设计的原则或要求，使用某些评价方法，对设计的过程和结果进行价值认定的活动。分为过程性评价和总结性评价，终结性评价是指对最终作品（产品）本身的总结性评价，是从技术的功能性、可靠性、创造性、文化性及专利保护等角度进行的全面而整体的评价。

【第13题】
武汉某高中将科技发明和专利法有关知识纳入校本教材，广泛开展“争当专利发明人”活动，该校很多学生申请专利，有的同学在教师的帮助下转让专利，并从中获得利益。下面关于专利权的说法正确的是（）

【第17题】
在设计的过程中，模型具有使设计对象具体化，帮助分析设计的可能性功能。某同学设计了一款老人用智能拐杖，想对其性能及人机关系进行研究，他应制作出（）

A　概念模型
B　草模
C　功能模型
D　结构模型
A B C D
答案：C
模型是根据实物、设计图纸，按比例制成的与实物相似的一种物体。草模是在设计初期，设计人员在构思草图的基础上，自己制作的能表达设计产品形态及基本体功能的模型。功能模型是验证产品功能设计合理性的模型，主要用于研究产品的一些物理性能、力学性能及人机关系。结构模型是能清晰地表达产品的结构尺寸和连接方法，并用于进行结构强度试验的模型。展示模型通常是在设计方案确定后，按所确定的形.态、尺寸、材质及表面效果等要求精细制作而成，外观与产品有相似的视觉效果，具有展示、宣传、设计交流与评价等作用。

【第19题】
无叶风扇由于没有传统风扇的罩子和扇叶,所以安全并易于清洗。因为没有扇叶“切割”空气,不会形成阶段性冲击和波浪形刺激,持续的空气气流会让人感觉更加自然凉爽。对该设计评价不合理的是( )



A　去掉传统风扇的罩子和扇叶，体现了设计的经济性原则
B　改变了传统风扇的出风方式,可以申请专利
C　改变了传统风扇外形结构， 属于结构创新
D　让人感觉更加自然凉爽，体现了良好的人机关系
A B C D
答案：A
本题考查的知识点主要是设计的评价，还有专利、人机关系、设计的一般原则。设计的评价可分为对设计过程的评价和对设计成果的评价。本题主要是对设计成果的评价，可从多方面进行，如功能、形态、效率、创新性、安全性、美学因素、易操作性、性价比，对社会、环境、资源的影响及市场应用前景等。去掉传统风扇的罩子和扇叶，并不一定降低了成本，也就不会体现经济性原则。


【第20题】
科技无止境。复旦大学研制成了具有二维半导体准”非易失”存储原型器件，开创了第三类存储技术，其写入速度比目前U盘快一万倍，这解决了国际半导体电荷存储技术中“写入速度"与"非易失性"难以兼得的难题。它得到了国家自然科学基全的支持，其相关成果发表于《自然．纳米技术》杂志。关于这个案例的分析不合理的是（）

A　研制"三类存储技术"，体现了技术的目的性
B　该技术得到国家自然科学基金支持，体现了技术的社会属性
C　研制一维半导体准"非易失"存储原型器件，属于技术活动
D　由于该技术成果发表于《自然．纳米技术》杂志，因此可获得专利保护

【第21题】
可充电LED台灯可以在停电的情况下依靠电池进行照明。下列从功能角度进行评价的是（）

A　色彩褡配合理，造型简约，让使用者赏心悦目
B　采用优质环保塑料制作，无毒无味，对人体无害
C　有触控式亮度调节开关，可调节不同亮度，适应不同环境
D　采用LED灯珠，发光效率高，节约电能，符合可持续发展原则
A B C D
答案：C
对最终产品的评价是对设计成果的总评价，因此要多方面把握评价内容，对一个目常用品，我们可以从功能、形态、效率、创新性、安全性、易操作性、性价比、美学，用对社会、环境、资源的影响以及市场应用前景等方面进行评价。当然，对不同的产品，评价的内容和标准可能会有所不同本题中，色彩搭配和造型是从外观美学角度进行评价；使用环保材料是从对环境的影响角度来评价；LED灯珠是从对资源的影响、可持续发展的角度来评价；只有触控式调节开关是从功能的角度进行评价的。
