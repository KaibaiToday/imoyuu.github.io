控制系统的组成和描述

【第1题】
有一种自动恒温干衣机可在烘干衣服时保持机内温度恒定，以免衣服被烧坏。其温度控制系统方框图如下。该控制系统的执行器是：（ ）



A　温度传感器
B　干衣机
C　控制电路
D　电热丝
A B C D
答案：D
控制系统中，输入量又时又是期望值，控制器是控制信号的发出者，执行器执行控制器发出的控制信号输出控制量作用于被控对象，被控对象是执行器作用的对象，被控量是被控对象的输出状态，对于闭环控制系统，组成部分还会多出一个反馈环节。 在本题执行器是电热丝。


【第2题】
如图所示是高速公路上普遍安装的超速自动抓拍控制系统，其工作原理是：雷达将检测到的信号送至控制主机进行处理和比较，如果发现超速车辆即启动摄像头拍照，照片将作为处罚的依据。请思考、回答下列题：该控制系统的输入量、输出量分别是（ ）



A　车辆速度 交警处罚
B　雷达 摄像头拍照
C　雷达 车辆速度
D　车辆速度摄 像头拍照
A B C D
答案：D
控制系统中，输入量又时又是期望值，控制器是控制信号的发出者，执行器执行控制器发出的控制信号输出控制量作用于被控对象，被控对象是执行器作用的对象，被控量是被控对象的输出状态，对于闭环控制系统，组成部分还会多出一个反馈环节。 在本题中输入量是雷达检测到的车辆速度，输出量是摄像头拍摄到的照片。?故选：D


【第3题】
消防管网的水压控制系统工作示意图如下。当消防管网压力降至设定下限时,水泵启动,使压力上升;当消防管网压力达到设定上限，水泵停止运行,使消防管网压力保持在设定的范围内。该控制系统的被控对象是( )



A　水压检测装置
B　控制单元
C　压力罐
D　水泵
A B C D
答案：C
根据闭环控制系统框图分析: 消防管网的水压控制系统，根据水位的高低，能自动启停，控制器是控制单元，执行器是水泵，被控对象是压力罐，水压检测装置属于检测反馈环节。


【第4题】
电热水器温度自动控制系统的构成示意图如下。当热电偶检测到箱内水温低于设定值时，控制电路控制加热丝工作，加热储水箱中的水，待水温达到被定值后，加热丝停止工作，使箱内水温保持在设定温度。下面关于该系统的说法正确的是()



A　被控对象是加热丝
B　执行器是热电偶
C　被控量是箱内水温
D　控制器是控制电路
这题是多选题
A B C D
答案：C,D
被控对象是控制系统中所要求控制的装置，本题被控对象是蓄水箱，被控量是控制系统所要控制的量，也是控制系统的输出信号，本题被控量是箱内温度，执行器是直接对被控对象进行控制的装置或元件，本题的执行器是加热丝，控制器是对输入信号进行处理并发出控制命令的装置或元件，本题的控制器是控制电路


【第5题】
以下各种控制系统中属于闭环控制的是()

A　学校楼道的声控灯系统
B　动车上的烟霎报警系统
C　家用空调的温度控制系统
D　酒店的自动门控制系统
A B C D
答案：C
判断一个系统是不是闭环控制系统，主要看它的输出量是否被反馈，并是否用来影响系统的输出。家用空调的温度控制系统简要工作过程:测温装置测量房间实际温度，将其转换为与设定温度量纲相同的信号并与之比较得到误差信号，误差放大装置将误差信号放大后去推动控温装置工作，使房间温度向设定温度趋近，直至误差被消除。因此，家用空调的温度控制系统属于闭环控制


【第6题】
有一款智能化的太阳能草坪灯，夜晚自动打开，白天灯会熄灭。在该控制系统中，被控对象是( )



A　灯
B　开关
C　光照
D　太阳能板
A B C D
答案：A
本题答案为：A。一个控制系统的工作，最终会以某种形式体现出控制的结果(输出量)，体现控制结果的环节就是这个控制系统的被控对象。这个控制系统工作的结果就是灯的亮灭，所以这个控制系统的被控对象是灯


【第7题】
抽水马桶的水箱水位控制系统属于( )



A　人工控制、闭环控制
B　自动控制、闭环控制
C　人工控制、开环控制
D　自动控制、开环控制
A B C D
答案：B
本题考査自动、人工和开环、闭环控制的区分。抽水马桶的水箱水位控制系统当水位低于设定水位时浮子下落,阀门打开,进水;浮子“监测”水面高度并随着睡眠升高而浮升。当水面达到设定的水位时,浮子传感,通过连杄机构将阀门关闭,这个过程是由人工控制可以用自动控制代替;由于阀门的开关由水位的偏差实现控制所以抽水马桶的水箱水位控制系统属于自动控制、闭环控制


【第8题】
如图所示是款新型太阳能饭盒，可以利用太阳能加热实现饭盒保温。该饭盒的保温控制系统中，被控量是( )



A　饭盒
B　太阳能
C　饭盒内实际温度
D　饭盒内的食物

【第9题】
蔬菜大棚可以让我们吃到反季蔬菜，它的湿度控制系统环控制系统，其工作原理是:传感器获取大棚内湿度数据，控制器根据实际湿度与设定湿度的差值，或开启排风循环扇降低湿度，或开启加湿杋增加湿度，直至达到设定值，从而保持大棚湿度恒定。判定该控制系统属于闭环控制的理由是( )



A　存在反馈环节
B　执行器是加湿器和风扇
C　控制电路能发出控制指
D　传感器能获取大棚内部的湿度值
A B C D
答案：A
由题目可知，蔬菜大棚湿度控制系统为闭环控制系统，与开环控制系统相比，闭环控制系统多了一个由监测装置组成的环节即存在反馈环节。


【第10题】
使用停车场智能车位显示系统能够快速、准确显示空余车位，控制系统方框图如下，该控制系统中的执行器是( )



A　信息发布芯片
B　空余车位
C　车位显示屏
D　探测车位信息
A B C D
答案：A
通过控制系统方框图可以判断本控制系统为开环控制系统。因此，识记开环系统方框图为本题的解题关键。探测车位信息为输入量: 信息发布芯片为执行器; 车位显示屏为被控对象 空余车位信息为输出量。


【第11题】
疲劳驾驶是绝大多数长途运输中事故的主要原因。有人设计的疲劳驾驶提酲系统的控制结构示意图如图所示,该提酲系统由摄像头、控制中心、提醒裝置三部分构成，首先通过摄像头摄取驾驶人员的人脸状态信息,通过控制中心与预设的疲劳状态的人脸特征比对,确定驾驶人员处于疲劳驾驶时通过继电器控制声音震动提醒装置发岀声音和震动。下列关于该控制系统的分析中正确的是()



A　执行器是继电器
B　被控对象是人
C　输人量是设定的疲劳状态人脸信息
D　控制量是声音和震动
A B C D
答案：A
本题考查系统的组成。被控对象是声音震动提醒装置;输入量是人脸状态信息;控制量是继电器的吸合。故选执行器是继电器。


【第12题】
汽车蓄电池是一个复杂的装置。汽车蓄电池电量控制系统示意图。汽玍电气负载消耗蓄电池的电能,当蓄电池电量下降到设定值下限时,发电机在动力控制模块的控制下给蓄电池充电,电量检测装置检测到蓄电池电量达到设定值上限时,发电机停止给蓄电池充电。下列关于该控制系统的分析中正确的是()



A　控制量是蓄电池的电量
B　汽车电气负载变化属于干扰因素
C　被控对象是汽车
D　执行器是动力控制模块
A B C D
答案：B
本题考查控制系统的组成及干扰因素。该系统目的是将蓄电池的电量控制在设定范围,被控对象为蓄电池,执行器为发电机,控制里为发电机的发电量,控制器为动力控制模块,输入量为蓄电池设定的电量,输出量为蓄电池的实际电量,汽车负载变化会影响输出量所以是千扰因素。

【第14题】
燃气热水器是某些家庭常用的热水装置，燃烧天然气加热进入热水器管道中的自来水。水温控制过程为∶温度传感器检测实际出水温度，若低于设定值，控制器加大天然气进气量加剧燃烧，使水温升高;若高于设定值，控制器减少天然气进气量，使水温降低，让水温保持在设定值。关于该控制系统的分析，以下说法正确的是（）

A　该控制系统是闭环控制系统
B　系统的输入量是天然气
C　温度传感器是检测装置，但不属于反馈环节
D　该控制系统构成比较复杂，属于抽象系统
A B C D
答案：A
本控制系统是闭环控制系统。我们把系统的输出量返回到输入端并对控制过程产生影响的控制系统称为闭环控制系统。本系统的检测装置是温度传感器，它将检测的实际水温返回到系统的输入端，属于反馈环节。本系统的输入量是设定水温。本系统是实物形态的，属于实体系统。
