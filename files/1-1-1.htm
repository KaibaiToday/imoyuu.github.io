第一章 技术及其巨大的作用

走进技术

【第1题】
智能手机应用了多项新技术，从技术的性质角度分析，以下说法中不正确的是(　　) A．具有通讯、播放、拍照、上网等功能，体现了技术的综合性 B．应用指纹识别技术提高安全性，体现了技术的创新性 C．长时间使用会伤害人的眼睛，体现了技术的两面性 D．厂商需要支付一定的技术使用费，体现了技术的专利性

A　具有通讯、播放、拍照、上网等功能，体现了技术的综合性
B　应用指纹识别技术提高安全性，体现了技术的创新性
C　长时间使用会伤害人的眼睛，体现了技术的两面性
D　厂商需要支付一定的技术使用费，体现了技术的专利性
A B C D
答案：A
技术的综合性指的是技术的跨学科性质。


【第2题】
高速路上都有ETC。ETC(高速公路电子收费系统)涉及了无线电通信、计算机、自动控制等多个知识领域。车辆在高速路出口驶入ETC车道时，系统对车辆身份进行识别,并从车辆IC卡中扣除相应通行费后,电动栏杆自动抬起,车辆即可通行。下列说法不正确的是?(?)

A　该系统专利属收费站所有
B　该系统收费使用”电子货币”体现创新性
C　电动栏杆控制系统解放了人
D　该系统体现了技术的综合性
A B C D
答案：A
解析：①ETC技术涉及无线电通信、计算机、自动控制等多个知识领域，体现了技术的综合性。 ②专利是国家按专利法授予申请人在一定时间内对其发明成果所享有的独占、使用和处分的权利。所以，ETC系统的专利权属于申请人。③ETC也叫不停车电子收费系统，是目前世界上最先进的路桥收费方式，使用电子货币是实现不停车收费的关键技术，非常具有创新性。④电动栏杆不需要人去抬起栏杆，解放了人。


【第3题】
下列属于技术范畴的是（）？

A　量子论
B　CT技术
C　万有引力的发现
D　相对论
A B C D
答案：B
解析：此题考察技术与科学的区别。技术强调实践，发明，解决“做什么，怎样做”:科学强调理论、发现，解决“是什么，怎么样，为什么”；科学在于认识世界，技术在于改造世界。万有引力的发现、相对论、量子论都是理论、是科学。CT技术是一种被称为X射线计算机断层扫描的技术。


【第4题】
手机二维码支付技术的快速发展,使人们购物更加便捷。同时也出现了个别商家拒绝现金支付的情况。这说明技术应用具有(?)



A　综合性
B　目的性
C　创新性
D　两面性
A B C D
答案：D
解析：任何事物在客观上都具有两面性，正面和负面。此题明显说的是技术应用的不好的方面，即负面影响。
