第三章 系统及其设计

认识系统

【第1题】
生活中系统随处可见,根据你的理解下列不属于系统的是( )

A　高层建筑物
B　桥墩
C　拦河大坝
D　一堆沙子和水泥
A B C D
答案：D
由相互联系、相互作用、相互依赖和相互制约的若干要素或部分组成的具有特定功能的有机整体，称为系统。比如高楼，台灯，桥墩，自行车，学校，公路，人体等。构成系统必须具备三个条件:第一，至少要有两个或者两个以上的要素(部分)才能组成系统; 第二，要素(部分)之间相互联系、相互作用、按照一定方式形成一个整体; . 第三，整体具有的功能是各个要素(部分)的功能中所没有的。 一堆沙子和水泥不属于系统。


【第2题】
汽车变速箱由许多齿轮组成;其中任何一个齿轮的损坏都会影响汽车的正常行驶，这说明系统具有( )

A　目的性
B　环境适应性
C　整体性
D　相关性
A B C D
答案：C
系统的基本特性有: 1.整体性:_强调要素与整体之间的影响及关系，表现在:①系统的整体功能是各要素在孤立状态下所没有的;②系统的任何一个要素发生变化或出现故障时,都会影响到整体功能的发挥。 2.相关性:强调系统内部要素与要素之间的相互作用、相互联系。 3.目的性:也称功能性，表现在任何系统都具有某种目的，都要实现一定的功能。 4.环境适应性:系统都存在于一定的环境中，并不断地与外界环境进行物质、能量和信息的交换。系统只有具有对环境的适应能力，才能保持和恢复系统原有的特性。 从题干来看，任何一个齿轮的损坏都会影响汽车的正常行驶，影响了汽车的整体性能的发挥。


【第3题】
2013年6月11日，神舟十号飞船从酒泉卫星中心发射中心发射升空，在轨飞行期间，飞船与天宫一号进行了交会对接，航天员王亚平在天宫一号开展了一系列空间科学实验，并进行了太空授课。从系统的角度分析，下列说法正确的是（）

A　天宫一号是一个抽象的开放系统
B　天官（天宫）一号是一个实体的开放系统
C　天宫一号是一个抽象的封闭系统
D　天宫一号是一个实体的封闭系统
A B C D
答案：B
按照系统组成元素存在的形态，把系统分为实体系统和抽象体统。由看得见、摸得着的物质实体为元素所构成的系统称为实体系统。由一些概念、原理、方法、制度、程序等元素构成的系统，称为抽象体统。按照系统与外部环境的关系来分，可分为开放系统和封闭系统。封闭系统是与外界无明显联系的系统。不断与外界环境进行信息、物质和能量的系统称为封闭系统。天宫一号是物质实体，虽然在太空中，和地面是时刻保持联系的，所以它是一个实体的开放的系统。


【第4题】
"挑战者号"航天飞机在进行第10次太空任务时，由于右侧固态火箭推进器里面的一个O形环失效，造成一连串的故障，在升空73秒后爆炸解体坠毁。这个方案要体现的系统的特性是（）

A　目的性
B　整体性
C　环境适应性
D　相关性
A B C D
答案：B
系统具有系统有这样几个基本特性:整体性、目的性、相关性、环境适应性、层次性。系统的整体性包括了两个方面。一是整体的功能是各个组成部分所没有的（例如:自行车的各个零部件都不具备行驶的功能，把它们按一定规律组合在一起就具备行驶的功能);二是系统不是各个组分（元素）的简单相加，系统内部的协调和配合状况直接影响整体功能的实现，即部分影响整体（例如:木桶理论、短板效应）。一个“O”形环失效，导致整体航天飞机的坠毁，体现了系统的整体性

【第8题】
春秋战国时期李冰设计建造的都江堰，具有分流、排沙、泄洪、灌溉等多种功能的综合性水利工程。该系统属于（）

A　抽象系统 自然系统
B　抽象系统 人工系统
C　实体系统 人工系统
D　实体系统 自然系统
A B C D
答案：C
本题考察的是系统的分类，由看得见、摸得着的具体实体为元素构成的系统称为实体系统。由一些概念、原理、方法、制度、程序等为元素所构成的系统称为抽象系统。由自然物自然形成的系统，称为自然系统。由人工建造、加工而构成的系统称为人工系统，也叫人造系统。都江堰属于实体系统，也属于人工系统。


【第9题】
太阳能光伏发电系统，可将太阳能转换为电能，并能储存电能.由光伏组件、控制器和蓄电池组等部分组成。从系统的角度理解，下列说法不正确的是（）



A　光伏组件能适应不同的气候条件，体现了系统的环境适应性
B　该系统实现了太阳能转化为电能，体现了系统的目的性
C　离网逆变器出现故障，导致某个交流负载损坏，体现了系统的相关性
D　电子线路会随使用时间的延长而老化，体现了系统的整体性
A B C D
答案：D
本题主要考察系统的四个基本特性。能够适应不同的气候条件，体现了系统适应外部环境的变化，这是系统的环境适应性，离网逆变器出现故障导致某个交流负载损坏，体现构成系统的元素之间的影响关系，这是系统的相关性，本系统的目的就是为了把太阳能转化为电能，体现了系统的目的性。 电子线路会随使用时间的延长而老化， 只能表明是物理环境和化学环境共同作用的结果，属于陈述事实，并不能体现系统的整体性。


【第10题】
款新型智能体感代步车，能根据人体姿势的变化自动实现前进、后退、转弯等操作，起到代步的作用。这主要体现了系统的( )



A　相关性
B　整体性
C　环境适应性
D　目的性
A B C D
答案：D
相关性，更加强调的是系统内部部分与部分的相互关系和相互作用，统一协调好能够起到事半功倍的效果；整体性，更加侧重的是系统整体和部分的关系，系统的整体功能是个部分在孤立状态下所没有的，起到1+1>2的作用; 环境适应性:强调的是外界的环境变化引起系统特性的改变，人体对于智能体感代步车来说是一个外部环境，但是外部环境人体姿势的改变并没有改变代步车的用来代步的特性。此题目的性和环境适应性不容易分辨，要理解系统系统的几个特性。


【第11题】
如图所示是一款太阳能草坪灯，根据系统分类，它属于( )



A　抽象系统
B　封闭系统
C　自然系统
D　人造系统
A B C D
答案：D
自然系统:是自然形成的系统，比如生态系统; 抽象系统是非实物形态的，比如存储在大脑中的知识结构; 封闭系统是与外界无明显联系的系统，如密闭罐中的化学反应。开放系统与外界有环境有信息、物质和能量交互作用，如生产系统，生态系统，太阳能草坪灯和外界有能量和信息的相互交换，应该是开放系统。


【第12题】
拿着手机走天下。如图所示是一款新型电动轮椅，可以通过手机控制方向与速度，使用非常方便。对手机和电动轮椅组成的这个系统，下列说法不合理的是（）



A　电动轮椅造型时尚，操作方便，体现了系统的整体性
B　电动轮椅组成部分相互影响，体现了系统的相关性
C　通过手机控制电动轮椅的方向与速度，体现了系统的目的性
D　电动轮椅在草地、石子路上也能如履平地，体现了系统的环境适应性
A B C D
答案：A
1、电动车轮椅面对草地，石子路进行系统感应而进行应变，符合系统对环境的适应性，所以正确。 2、系统的整体性指系统是作为一个由诸多要素结合而成的有机整体存在并发挥作用的，简单讲就是系统具有其部分在孤立状态下所没有的整体特性，而题目中并没有提及到产品的整体与部分的阐述，看不出整体性的表述，所以错误。 3、相关性是指组成系统的各元素之间或系统与部分之间的相互作用、相互联系、相互依赖和相互制约。题目中提到相互影响所以正确。 4、系统的基本特征一目的性，系统的目的性，也称为功能性。
