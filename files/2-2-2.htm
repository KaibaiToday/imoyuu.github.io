第二章 流程及其设计

流程的组成与描述

【第1题】
下列有关流程的说法,错误的是（ ）

A　不能颠倒时序的环节说明事物内在之间有一定的规律和机理
B　维修电器前先断电,是为了保障安全
C　在氢气燃烧的实验中,只要氢气制取过程严谨,验纯环节可以省略
D　生产流程中各环节之间的时序都不能颠倒
A B C D
答案：D
解析： 根据情况,生产流程中各环节之间的时序有的可以颠倒,有的不能颠倒。


【第2题】
某学校饭卡自助充值机的充值环节包括:①点击确认按钮,确认充值金额及饭卡信息;②插入饭卡,点击“饭卡充值";③选择充值金额;④审核充值金额及饭卡信息;⑤选择充值方式并付款,完成饭卡自助充值,下列关于该充值流程的描述不恰当的是( )

A　自助充值机进行饭卡充值的流程可分为五个环节
B　输入充值金额和确认充值金额两个环节,时序不可颠倒
C　该饭卡自助充值机合理的充值流程是②③①④⑤
D　与人工充值相比,自助充值机提高了充值效率

【第3题】
下图所示为某单位高纯度水的制作流程示意图。对该流程的分析正确的是( )

A　清洗用水从二级反渗透环节后取用,会提高生产成本
B　紫外线灯与蒸馏器是串行环节
C　纯化水经紫外线灯消毒,可以用作注射用水
D　预过滤环节与精滤环节的时序可颠倒
A B C D
答案：A
解析: 考察技术素养(工程思维),学科内容(流程图的识读、流程的设计、流程的优化)这是一道综合题,时序是指任何一项生活或生产活动的时间上的顺序。时序是否可颠倒,是根据事物本身内在的机理来决定的。环节是指对于一项活动,我们可以依据它的某种特征或某个具体目标,将其分解为若干个过程或阶段。串行工序是指每个环节都依次进行的工序,并行工序是指工序是同时进行的。

【第5题】
下列关于流程的描述,正确的是( )

A　流程的表达通常用流程图,其中方框流程图是最常用的
B　任何流程环节的先后顺序(时序)都是可以调整的
C　两个或两个以上的环节就能构成流程
D　任何流程都能反映事物的合理的内在机理
A B C D
答案：A
解析： 考察技术素养(工程思维),学科内容(流程的含义)时序是否可以颠倒,是根据事物本身内在的机理来决定的。如果后一项任务是以前一项任务的完成为条件的,或者一项任务的完成会对另一项任务生较大的负面影响时,时序就不能颠倒。我们应尊重和遵循这些规律或机理。任何流程都反映了一定的时序,体现岀一定的环节。依据流程的性质以及人类的表达习惯,流程可以通过文字、表格、图示等方式来表示,有些场合还可以通过模型、动画等方式来表达。


【第6题】
高压锅的历史悠久， 生活中使用高压锅烹饪食物 ，出锅前要先释放锅内蒸汽再开锅盖， 否则会发生危险。以上事例说明（ ）



A　流程的时序都是不可颠倒的
B　环节划分反映了事物的内在规律
C　时序具有相对性，可以自由安排
D　合理的时序可以为我们的安全提供保障
A B C D
答案：D
解析： 本题考查时序与环节、生活与流程。科学合理的流程，可以指导我们正确地做事，提高工作和学习的效率，使我们的生活变得有序、合理，为我们的安全提供保障;时序体现了具体活动内容的先后关系，在这种先后关系中，有些步骤之间的时序是不可颠倒的。使用高压锅时要遵循事物的规律，不能随意颠倒时序;说明生活中处处有流程，合理正确的流程可以引导我们正确做事，保障我们的安全。故正确选项是“合理的时序可以为我们的安全提供保障”。


【第7题】
为使流程的描述清晰可见， 我们一般可以采用多种表达方式。 下列不属于流程表达方式的是( )

A　文字表达
B　表格表达
C　图示表达
D　三视图表达
A B C D
答案：D
解析： 根据流程的性质以及人们的表达习惯，流程可以通过文字、表格、图示等方式表达，有些场合还可以利用模型、动画等当时表达。丕论哪种方式，都包含时序和环节，表达出环节进行的时序。选项中三视图是用来表达物品设计效果的，不是流程图。
