【第2题】
如图所示的箭靶架，两个支撑脚之间采用铰连接。箭靶中箭时，易翻倒，下列改进方案中最合理的是（）



A B C D
答案：C

【第4题】
图示为用硬纸板制作而成的座椅，具有一定的强度，能承受一个成年人的重量。以下不属于影响纸板座椅强度的主要因素是（）





A　硬纸板与硬纸板之间的连接方式
B　硬纸板连接处的结构
C　纸板座椅座面的高度
D　硬纸板的强度
A B C D
答案：C
技术素养(创新设计、物化能力),学科内容(结构的强度)。 结构抵抗荷载的能力用强度来表示。结构强度是指结构具有的抵抗被外力破坏的能力。当结构构件受到外力作用时，内部各质点间的相互作用发生改变，产生一种抵抗外力与形变的力，称为内力。影响结构强度的因素有结构的形状(包括截面形状)、连接方式及材料的选择等。改变座椅座面的高度，就改变了结构的重心，会影响结构的稳定

A　台灯

【第8题】
分析下列物体的结构类型，属于实体结构的是（）

A　电灯泡
B　乒乓球
C　橡皮擦
D　篮球
A B C D
答案：C
根据结构的力学架构形态可分为：实体结构、框架结构、壳体结构。 A、B、D选项均属于壳体结构，C选项属于实体结构。


【第9题】
如图所示是一座简易木质桥梁模型，现在想对其进行加固，下列方案不合理的是



A B C D
答案：C
铁丝只能承受拉力，C方案用铁丝连接中间的立杆，铁丝无法给桥梁向上的拉力，与题干图相比桥面承重反而增加了两个立柱的重量，使结构更不稳固，故选C。
