【第1题】
汽车安全气囊控制系统的工作过程是：传感器检测到汽车碰撞信号，经诊断模块判定和处理后，触发气体发生器给气囊充气。该控制系统的被控对象是（）

A　被气囊保护的人
B　诊断模块
C　气囊
D　气体发生器
A B C D
答案：C
控制系统中，输入量又时又是期望值，控制器是控制信号的发出者，执行器执行控制器发出的控制信号输出控制量作用于被控对象，被控对象是执行器作用的对象，被控量是被控对象的输出状态，对于闭环控制系统，组成部分还会多出一个反馈环节。 本系统没有反馈环节属于开环控制系统。 A选项：被气囊保护的人是汽车的使用者，不属于安全气囊控制系统的一部分，错误； B选项：诊断模块处理传感器检测到的碰撞信息，后作出判断，属于控制器，错误； D选项：气体发生器直接给气囊充气，使其能弹出，属于执行器，错误； C选项：当检测到碰撞信号后，安全气囊控制系统给气囊充气，让其弹出，故气囊是被控对象，故正确。故选：C。

【第7题】
智能家居将是末来生活的主题。如图所示是红外感应马桶原理图。当红外线传感器检测到人体信号后输送给微电脑,微电脑接收到信号进行处理并发送给脉冲电磁阀,脉冲电磁阀接收信号后按指定的指令打开控制进水口出水。当人们离开时红外线感应器将信号传递给微电脑,微电脑根据信号,控制冲水装置排放适当的水量。该马桶的电磁阀控制子系统和冲水子系统的被控对象分别是()



A　马桶、水箱
B　电磁阀、冲水装置
C　红外传感器、马桶
D　电磁阀、马桶
A B C D

【第8题】
智能化的应用无处不在。下图是一款带显示屏的智能花盆,内置多种传感器和微处理芯片。用户手机与花盆间通过APP联网后,可手动设置基本参数等初始信息。花盆测得土壤的水分、光照和温度等,与设置好的参数进行比较,即可在屏幕上显示对应的动画,使花盆更加人性化。用户可通过APP监护,使植物保持活力。下列关于显示屏控制子系统的说法中合理的是()



A　内置传感器是反馈用的检测装置
B　用户通过APP重新设置参数属于干扰因素
C　所显示的动画与检测的结果相对应,所以是闭环控制
D　被控对象是显示屏
A B C D
答案：D
1此显示屏控制子系统的输出量是动画,而动画并不会对输入量一土壤的水分等信息产生影响,所以,是开环控制。当出现动画后,系统不会自动去做什么来改善土壤环境,是需要人工去做,所以,该控制系统是开环控制系统。2.内置传感器是检测装置,但不是反馈用的,是对土壤环境进行对比检测后,作为输入量来启动此开环控制过程的。3.APP设置参数是为了得到实际土壤环境和应有土壤环境的差别,作为输入量的,是必须的数据,所以不属于干扰因素。

【第12题】
某款豆浆机的保温控制系统框图如下。对该控制系统分析不正确的是( )





A　被控对象是豆浆机
B　该控制系统具有反馈环节
C　检测装置检测的是给定温度
D　执行器是加热管
这题是多选题
A B C D
答案：C
本题答案为：C。对应闭环控制系统的一般框图可以看出:执行器是加热管，被控对象是豆浆机；从题中可以看出此控制系统的反馈环节就是检测装置；但是检测装置检测并反馈的是输出量也就是实际温度
